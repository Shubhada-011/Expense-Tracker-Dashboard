# Personal Expense Tracker Dashboard (Microsoft Excel)

## Project Overview

This project is an interactive Microsoft Excel dashboard developed to analyze personal spending patterns and provide actionable financial insights. The dashboard transforms raw expense records into meaningful visualizations, helping users monitor expenses, identify major spending categories, and track monthly spending trends.

---

## Business Problem

Managing personal expenses can be difficult when transactions are spread across multiple categories and dates. Without proper analysis, it becomes challenging to understand where money is being spent and identify opportunities for savings.

This dashboard was created to provide a centralized view of spending and support better financial decision-making.

---

## Objectives

* Analyze total expenses across different categories.
* Monitor monthly spending trends.
* Compare spending across expense categories.
* Provide interactive filtering using slicers.
* Present key financial metrics through KPI cards.
